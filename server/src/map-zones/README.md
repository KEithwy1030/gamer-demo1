# 分区地图系统

> 这是流荒之路的"分区地图系统"板块。
> 整体架构见 `docs/REFACTOR-GUIDE.md`。
> 修改此板块代码前必读：本 README + REFACTOR-GUIDE 第一部分（产品铁则）。

## 状态

新独立（合并 关卡布局 + 多地图管理）

## 负责什么

- 一张大地图（9600×9600 或更大）分成 4-6 个有显著主题差异的战区
- 每个战区的特征定义（怪物密度、掉落质量、地形危险、宝箱密度）
- 玩家在大厅选择出生战区（不同战区 → 不同难度）
- 撤离点位置（2-3 个，分布在中央）
- 地图静态布局（出生点、安全区、河流、宝箱位置）

## 不负责什么

- 怪物的具体生成行为（怪物系统）
- 宝箱的具体掉落内容（宝箱系统）
- 河流的实际伤害判定（环境系统）
