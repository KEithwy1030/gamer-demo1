# 测试反馈问题列表

## Issue 1: 移动端行动摇杆不连贯且大幅拖动易断触
- 描述：移动端虚拟摇杆的连续输入体验异常；当摇杆拖动幅度较大时更容易出现断触，导致移动方向和速度不稳定。
- 复现步骤：
  1. 在手机端进入游戏对局。
  2. 持续拖动左侧虚拟摇杆进行移动，并尝试较大幅度拖动。
  3. 观察角色移动的连续性、方向响应和触控保持情况。
- 期望行为：摇杆拖动期间，角色移动方向与速度连续、稳定；即使较大幅度拖动也不应断触或频繁归零。
- 实际行为：行动摇杆不连贯，且拖动幅度过大时容易断触。

## Issue 2: 移动端右侧操作按键整体无效
- 描述：移动端右侧操作区按键整体不生效，无法稳定触发对应行为，影响核心交互与战斗操作。
- 复现步骤：
  1. 在手机端进入游戏对局。
  2. 尝试点击右侧各类操作按钮。
  3. 在可交互对象附近重复测试。
- 期望行为：右侧操作按钮应稳定触发对应行为，例如攻击、技能、拾取、交互或撤离。
- 实际行为：手机端右边按键都不生效。

## Issue 3: 手机端移动速度需要减半
- 描述：移动端虚拟摇杆驱动下的角色移动速度明显偏快，当前测试要求直接将手机端移动速度减半。
- 复现步骤：
  1. 在手机端进入游戏对局。
  2. 使用虚拟摇杆持续移动角色。
  3. 对比当前移动节奏与测试预期。
- 期望行为：手机端角色移动速度在当前基础上减半，降低触控端速度优势并改善操作可控性。
- 实际行为：人物移动速度太快。

## Issue 4: 刀剑枪攻击反馈区分度不足
- 描述：当前三种武器的攻击表现差异不明显，玩家无法清晰感知刀、剑、枪的攻击方式区别。
- 复现步骤：
  1. 在游戏内分别使用刀、剑、枪进行攻击。
  2. 观察攻击动作、攻击轨迹、命中视觉反馈。
  3. 对比三种武器的攻击表现差异。
- 期望行为：
  - 刀应体现劈砍感。
  - 剑应体现戳刺感。
  - 枪应体现挥舞感。
  三种武器在攻击动作表现、命中视觉反馈或攻击轨迹上应有明显区分。
- 实际行为：攻击反馈区分不出武器之间的差别，当前手感与视觉表现过于同质化。

## Issue 5: 装备界面显示不完全
- 描述：装备界面在当前测试设备上显示不完整，存在内容被截断或显示区域不足的问题，影响装备信息查看与操作。
- 复现步骤：
  1. 进入游戏并打开装备/背包相关界面。
  2. 观察装备面板整体显示范围。
  3. 检查装备槽位、文本或操作区域是否完整可见。
- 期望行为：装备界面应在当前分辨率/设备下完整显示，关键信息与操作区域可见且不被裁切。
- 实际行为：装备界面显示不完全。

## Issue 6: 玩家基础攻击距离需要再次翻倍
- 描述：在已有反馈基础上，当前玩家基础攻击距离仍然不足，测试要求在现有基础上再次翻倍。
- 复现步骤：
  1. 在游戏内使用当前武器进行普通攻击。
  2. 观察角色与目标之间的有效命中距离。
  3. 对比当前覆盖范围与测试预期。
- 期望行为：玩家当前基础攻击距离在现有版本基础上再次翻倍，进一步扩大攻击覆盖范围。
- 实际行为：当前攻击距离仍然偏短。
## Issue 7: 刀与枪的技能需要补齐，并增加高伤害与冷却机制
- 描述：当前刀和枪的技能仍未实装完整；测试要求补齐技能，并统一加入高伤害与冷却机制。
- 复现步骤：
  1. 在游戏内切换到刀或枪。
  2. 尝试释放对应技能。
  3. 观察技能是否存在、是否可触发、是否造成有效伤害，以及是否存在冷却。
- 期望行为：
  - 刀与枪都应具备可正常释放的技能；
  - 技能伤害提升到当前预期的 10 倍量级；
  - 技能加入冷却机制，初始冷却时间为 3 秒。
- 实际行为：刀和枪的技能仍然没有，且缺少对应的高伤害与冷却设计。
## Issue 8: 怪物伤害需要减半
- 描述：当前怪物输出伤害偏高，影响战斗节奏与前期容错。
- 复现步骤：
  1. 在游戏内与怪物交战。
  2. 观察怪物单次攻击对玩家造成的伤害。
  3. 对比测试预期的生存压力。
- 期望行为：怪物造成的伤害应在当前基础上减半。
- 实际行为：怪物伤害过高。
## Issue 9: 怪物攻击距离需要减半
- 描述：当前怪物攻击判定距离偏长，影响走位空间与近身交互合理性。

  2. 观察怪物触发攻击的距离范围。

## Issue 10: 宝箱系统缺失
- 描述：当前游戏中未实现宝箱系统，缺少对应的场景交互与奖励承载内容。
- 复现步骤：
  1. 进入游戏进行探索。
  2. 观察地图内可交互资源点与搜刮内容。
  3. 确认是否存在宝箱相关对象、开启交互或掉落反馈。
- 期望行为：游戏中应具备基础宝箱系统，至少包含宝箱对象、开启交互，以及开启后的掉落/奖励反馈。
- 实际行为：宝箱系统没有。

## Issue 11: 攻击缺少伤害数值与爽感反馈
- 描述：当前攻击命中后缺少即时数值反馈和打击表现，玩家难以感知命中结果，战斗缺乏爽感。
- 复现步骤：
  1. 在游戏内使用武器攻击怪物或目标。
  2. 观察命中瞬间的视觉反馈。
  3. 检查是否出现伤害数值、命中特效或其他强化打击感的表现。
- 期望行为：攻击命中后应显示清晰的伤害数值，并补充基础打击反馈元素（如命中表现、受击反馈、数值跳字等），增强战斗爽感。
- 实际行为：攻击之后没有任何明显的数值或爽感元素显示。

## Issue 14: 怪物需要增加攻击效果
- 描述：当前怪物攻击缺少足够明确的攻击表现，玩家较难第一时间感知怪物出手与命中威胁。
- 复现步骤：

  2. 观察怪物攻击前摇、攻击动作、攻击特效或命中表现。

- 实际行为：怪物攻击效果不足。
## Issue 15: 玩家与怪物需要增加被攻击反馈
- 描述：当前玩家与怪物在受到攻击后缺少明显受击反馈，影响命中确认和战斗打击感。
- 复现步骤：
  1. 在游戏内让玩家攻击怪物，或让怪物攻击玩家。
  2. 观察受击目标在命中后的反馈表现。
  3. 检查是否存在受击闪烁、击退、受击动作、命中特效或其他反馈元素。
- 期望行为：玩家与怪物在被攻击时都应具备清晰的受击反馈，以增强命中确认、可读性与战斗表现。
- 实际行为：玩家和怪物的被攻击反馈不足。

## Issue 16: 整体战斗体验需向“丛林肉搏感”靠拢
- 描述：当前游戏需要在整体体验上向《魔兽争霸 RPG：丛林肉搏》靠拢，重点不是复刻玩法规则，而是借鉴其战斗成长与爽感结构。
- 复现步骤：
  1. 体验当前版本的刷怪、战斗、掉落与成长循环。
  2. 对比“丛林肉搏”式体验中常见的成长节奏和战斗反馈。
  3. 评估当前版本在爽感、成长感和节奏感上的差距。
- 期望行为：当前版本后续优化应优先向以下方向靠拢：
  - 强化刷怪成长感；
  - 强化攻击打击感、伤害跳字和爽感反馈；
  - 强化掉宝与装备成长体验；
  - 强化技能爆发感；
  - 提升怪物密度与清怪节奏；
  - 让整体战斗节奏更简单直接、上手即爽。
- 实际行为：当前版本尚未形成明确的“丛林肉搏式”战斗体验。
