## P1 体验问题（能玩，但体验差）

### Issue 19: 缺少回血手段
- 描述：当前游戏内缺少明确可用的回血机制，导致生存恢复链路缺失。
- 复现步骤：
  1. 进入游戏并在战斗中受伤。
  2. 尝试通过现有系统寻找回血或恢复生命值的方式。
  3. 观察是否存在稳定可用的回复手段。
- 期望行为：游戏内应提供至少一种明确的回血手段，例如回复道具、技能回复、脱战恢复或其他可执行恢复机制。
- 实际行为：当前没有回血手段。

### Issue 22: 其他品质装备设计已存在，但实际没有掉落
- 描述：当前装备掉落实际上只有白装，已设计的其他颜色品质装备没有正常掉落，导致品质成长系统未生效。
- 复现步骤：
  1. 在游戏内持续击杀怪物并观察掉落。
  2. 检查掉落装备的品质颜色与词条数量。
  3. 确认是否能实际获得白色以外的装备。
- 期望行为：除白装外，应实际掉落其他颜色品质装备；并且每提升一个品质等级，就增加一个词条，形成明确的品质成长层级。
- 实际行为：当前只有白色装备掉落，设计中的其他品质装备没有实际掉落。

### Issue 27: 怪物需要打散分布到各个区域
- 描述：当前怪物分布方式不符合测试预期，需要把怪物更均匀地打散到地图各区域。
- 复现步骤：
  1. 进入地图探索并观察怪物刷新位置。
  2. 检查怪物是否过度集中在少数区域。
- 期望行为：怪物应打散分布在地图各个区域，提升探索感、分区战斗节奏与资源分布合理性。
- 实际行为：当前怪物分布集中度过高或区域覆盖不足。

### Issue 28: 精英怪数量需要减少一半
- 描述：当前精英怪数量偏多，不符合测试预期。
- 复现步骤：
  1. 进入地图探索并统计精英怪出现频率与分布。
  2. 对比普通怪与精英怪的体感比例。
- 期望行为：精英怪数量在当前版本基础上减少一半。
- 实际行为：当前精英怪数量偏多。

### Issue 29: 玩家出生时应自带一把基础武器，出生点不再放其他物品
- 描述：当前出生点资源配置不符合测试预期；最新要求不是在出生点额外堆放武器，而是让玩家出生时直接自带一把基础武器，同时出生点本身不再放其他东西。
- 复现步骤：
  1. 创建或进入一局新游戏。
  2. 观察角色出生后的初始持有状态。
  3. 检查出生点周围是否仍然摆放额外武器或其他物品。
- 期望行为：每个玩家出生时直接自带一把基础武器；出生点本身不额外放置其他武器或物品。
- 实际行为：当前出生配置不符合该目标。

### Issue 30: 怪物死亡后尸体需在 10 秒后消失
- 描述：当前怪物死亡后的尸体残留时间不符合测试要求，需要明确清理时长。
- 复现步骤：
  1. 在游戏内击杀怪物。
  2. 观察怪物尸体在场景中的保留时间。
- 期望行为：怪物死亡后尸体在 10 秒后消失。
- 实际行为：当前尸体消失时长不符合该要求。

### Issue 31: 怪物站位需分散且全随机
- 描述：当前怪物站位与分布方式不符合测试预期，需要分散化并引入全随机站位逻辑。

- 期望行为：怪物在地图内分散站位，并采用全随机分布策略，避免过度集中和固定化。
- 实际行为：当前怪物站位不够分散或随机性不足。

### Issue 32: 怪物死亡后 1 分钟刷新新怪物

- 复现步骤：
  1. 在游戏内击杀怪物。
  2. 记录原区域或对应刷新逻辑下的新怪生成时间。
- 期望行为：怪物死亡后 1 分钟刷新新的怪物。
- 实际行为：当前怪物刷新时间不符合该要求。

### Issue 33: 怪物掉落率需调整为 50%，并提高高品质装备掉落率
- 描述：当前怪物掉落收益不符合测试预期，需要同时调整基础掉落率与高品质装备产出概率。
- 复现步骤：
  1. 在游戏内持续击杀怪物。
  2. 统计怪物掉落触发频率与装备品质分布。

- 实际行为：当前掉落率和高品质装备产出率偏低，不符合最新测试预期。

- 期望行为：玩家当前攻击距离在现有版本基础上减少 50%，让攻击判定更贴近合理近战范围。
- 实际行为：当前攻击距离过长，不符合最新测试预期。

### Issue 7: 刀与枪技能需要按当前目标重新校准，并让攻击动画适配攻速
- 描述：刀与枪技能虽已做过一轮实现，但当前测试要求继续按最新战斗节奏重做/重校；同时剑的攻击节奏和动画也要适配攻速变化。
- 复现步骤：
  1. 在游戏内分别使用刀、枪、剑进行普通攻击和技能释放。
  2. 观察刀与枪的技能可用性、伤害表现与冷却节奏。
  3. 观察剑在基础攻速与攻速提升后的攻击动画表现是否与频率一致。
- 期望行为：
  - 剑基础攻速按 1.5 次/秒作为目标节奏；
  - 攻速属性继续提高攻击频率；
  - 攻击动画必须适配攻击速度变化，避免动画节奏和实际攻击频率脱节；
  - 刀与枪技能按当前测试目标继续补齐和校准。
- 实际行为：当前刀与枪技能仍不符合测试预期，攻击动画也未充分适配攻速变化。

## 设计改版 / 新功能（先拍板再做，不要和修 bug 混在一起）

### Issue 16: 整体战斗体验需向"丛林肉搏感"靠拢

- 复现步骤：
  1. 体验当前版本的刷怪、战斗、掉落与成长循环。
  2. 对比“丛林肉搏”式体验中常见的成长节奏和战斗反馈。
  3. 评估当前版本在爽感、成长感和节奏感上的差距。
- 期望行为：
  - 强化刷怪成长感
  - 强化攻击打击感、伤害跳字和爽感反馈
  - 强化掉宝与装备成长体验
  - 强化技能爆发感
  - 提升怪物密度与清怪节奏
