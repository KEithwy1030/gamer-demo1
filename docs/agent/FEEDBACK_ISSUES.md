## Issue 16: 整体战斗体验需向"丛林肉搏感"靠拢
- 描述：当前游戏需要在整体体验上向《魔兽争霸 RPG：丛林肉搏》靠拢，重点不是复刻玩法规则，而是借鉴其战斗成长与爽感结构。
- 复现步骤：
  1. 体验当前版本的刷怪、战斗、掉落与成长循环。
  2. 对比“丛林肉搏”式体验中常见的成长节奏和战斗反馈。
  3. 评估当前版本在爽感、成长感和节奏感上的差距。
- 期望行为：
  - 强化刷怪成长感
  - 强化攻击打击感、伤害跳字和爽感反馈
  - 强化掉宝与装备成长体验
  - 强化技能爆发感
  - 提升怪物密度与清怪节奏
  - 让整体战斗节奏更简单直接、上手即爽
- 实际行为：当前版本尚未形成明确的“丛林肉搏式”战斗体验。
- 状态：设计方向，持续迭代中。
