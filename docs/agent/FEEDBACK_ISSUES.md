# 测试反馈问题列表

## Issue 1: 移动端行动摇杆不连贯且大幅拖动易断触
- 描述：移动端虚拟摇杆的连续输入体验异常；当摇杆拖动幅度较大时更容易出现断触，导致移动方向和速度不稳定。
- 复现步骤：
  1. 在手机端进入游戏对局。
  2. 持续拖动左侧虚拟摇杆进行移动，并尝试较大幅度拖动。
  3. 观察角色移动的连续性、方向响应和触控保持情况。
- 期望行为：摇杆拖动期间，角色移动方向与速度连续、稳定；即使较大幅度拖动也不应断触或频繁归零。
- 实际行为：行动摇杆不连贯，且拖动幅度过大时容易断触。

## Issue 2: 移动端右侧操作按键整体无效
- 描述：移动端右侧操作区按键整体不生效，无法稳定触发对应行为，影响核心交互与战斗操作。
- 复现步骤：
  1. 在手机端进入游戏对局。
  2. 尝试点击右侧各类操作按钮。
  3. 在可交互对象附近重复测试。
- 期望行为：右侧操作按钮应稳定触发对应行为，例如攻击、技能、拾取、交互或撤离。
- 实际行为：手机端右边按键都不生效。

## Issue 3: 手机端移动速度过快
- 描述：移动端虚拟摇杆驱动下的角色移动速度偏快，影响操作手感、节奏和端间平衡。
- 复现步骤：
  1. 在手机端进入游戏对局。
  2. 使用虚拟摇杆持续移动角色。
  3. 对比桌面端或正常预期下的角色移动节奏。
- 期望行为：移动端角色移动速度应与桌面端手感和游戏节奏基本一致，不应获得异常速度优势。
- 实际行为：人物移动速度太快。

## Issue 4: 刀剑枪攻击反馈区分度不足
- 描述：当前三种武器的攻击表现差异不明显，玩家无法清晰感知刀、剑、枪的攻击方式区别。
- 复现步骤：
  1. 在游戏内分别使用刀、剑、枪进行攻击。
  2. 观察攻击动作、攻击轨迹、命中视觉反馈。
  3. 对比三种武器的攻击表现差异。
- 期望行为：
  - 刀应体现劈砍感。
  - 剑应体现戳刺感。
  - 枪应体现挥舞感。
  三种武器在攻击动作表现、命中视觉反馈或攻击轨迹上应有明显区分。
- 实际行为：攻击反馈区分不出武器之间的差别，当前手感与视觉表现过于同质化。

## Issue 5: 装备界面显示不完全
- 描述：装备界面在当前测试设备上显示不完整，存在内容被截断或显示区域不足的问题，影响装备信息查看与操作。
- 复现步骤：
  1. 进入游戏并打开装备/背包相关界面。
  2. 观察装备面板整体显示范围。
  3. 检查装备槽位、文本或操作区域是否完整可见。
- 期望行为：装备界面应在当前分辨率/设备下完整显示，关键信息与操作区域可见且不被裁切。
- 实际行为：装备界面显示不完全。

## Issue 6: 玩家基础攻击距离需要翻倍
- 描述：当前玩家基础攻击距离偏短，影响战斗手感与命中体验，需要整体上调。
- 复现步骤：
  1. 在游戏内使用当前武器进行普通攻击。
  2. 观察角色与目标之间的有效命中距离。
  3. 对比测试预期的攻击覆盖范围。
- 期望行为：当前基础攻击距离应在现有数值基础上翻倍，以提升近战/武器攻击的覆盖范围和手感。
- 实际行为：当前攻击距离过短，不符合测试预期。
## Issue 7: 刀与枪的技能需要实装
- 描述：当前刀和枪的技能尚未完整实装，导致武器体验不完整。
- 复现步骤：
  1. 在游戏内切换到刀或枪。
  2. 尝试使用对应技能。
  3. 观察技能是否存在、是否可触发、是否有实际效果。
- 期望行为：刀与枪应具备已设计的技能，并可在游戏内正常触发与生效。
- 实际行为：刀和枪的技能需要补充实装。
## Issue 8: 怪物伤害需要减半
- 描述：当前怪物输出伤害偏高，影响战斗节奏与前期容错。
- 复现步骤：
  1. 在游戏内与怪物交战。
  2. 观察怪物单次攻击对玩家造成的伤害。
  3. 对比测试预期的生存压力。
- 期望行为：怪物造成的伤害应在当前基础上减半。
- 实际行为：怪物伤害过高。
## Issue 9: 怪物攻击距离需要减半
- 描述：当前怪物攻击判定距离偏长，影响走位空间与近身交互合理性。
- 复现步骤：
  1. 在游戏内靠近怪物并尝试拉扯走位。
  2. 观察怪物触发攻击的距离范围。
  3. 对比测试预期的攻击判定距离。
- 期望行为：怪物攻击距离应在当前基础上减半。
- 实际行为：怪物攻击距离过长。

## Issue 10: 宝箱系统缺失
- 描述：当前游戏中未实现宝箱系统，缺少对应的场景交互与奖励承载内容。
- 复现步骤：
  1. 进入游戏进行探索。
  2. 观察地图内可交互资源点与搜刮内容。
  3. 确认是否存在宝箱相关对象、开启交互或掉落反馈。
- 期望行为：游戏中应具备基础宝箱系统，至少包含宝箱对象、开启交互，以及开启后的掉落/奖励反馈。
- 实际行为：宝箱系统没有。

## Issue 11: 攻击缺少伤害数值与爽感反馈
- 描述：当前攻击命中后缺少即时数值反馈和打击表现，玩家难以感知命中结果，战斗缺乏爽感。
- 复现步骤：
  1. 在游戏内使用武器攻击怪物或目标。
  2. 观察命中瞬间的视觉反馈。
  3. 检查是否出现伤害数值、命中特效或其他强化打击感的表现。
- 期望行为：攻击命中后应显示清晰的伤害数值，并补充基础打击反馈元素（如命中表现、受击反馈、数值跳字等），增强战斗爽感。
- 实际行为：攻击之后没有任何明显的数值或爽感元素显示。

## Issue 12: iOS 端无法移动
- 描述：游戏在 iOS 手机端存在角色无法移动的问题，移动端核心操作链路失效。
- 复现步骤：
  1. 使用 iOS 手机打开游戏。
  2. 进入游戏对局。
  3. 尝试通过虚拟摇杆控制角色移动。
- 期望行为：iOS 端应能通过虚拟摇杆正常控制角色移动。
- 实际行为：iOS 端不能移动。
